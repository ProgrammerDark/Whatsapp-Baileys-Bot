# WhatsApp easy-baileys Bot

A WhatsApp bot built with [easy-baileys]

## Features

- Send and receive messages
- Handle group chats
- Automated responses
- Message logging

## Installation

```bash
yarn install
```

## Configuration

Create a `.env` file with your configuration:

```
MONGOD_URI=
```

## Usage

```bash
yarn start
```

## Requirements

- Node.js >= 20
- yarn or npm
